# GharTk - One App for Everything

## Project Overview
**GharTk** is a modern, high-performance "Super App" designed to provide a seamless experience for food delivery, grocery shopping, and ride booking. Built with React and Vite, it features a fluid interactive UI, responsive design, and a robust cart management system.

---

## Technologies Used
- **Frontend**: React 19, Vite
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM 7
