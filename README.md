# 🚀 GharTk - The Ultimate Super App

[![TailwindCSS](https://img.shields.io/badge/tailwindcss-%2338B2AC.svg?style=for-the-badge&logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)
[![Framer Motion](https://img.shields.io/badge/Framer_Motion-0055FF?style=for-the-badge&logo=framer&logoColor=white)](https://www.framer.com/motion/)

**GharTk** is a premium, high-performance Super App platform that seamlessly integrates **Food Delivery**, **Grocery Shopping**, and **Ride Booking** into a single, cohesive user experience. Designed with a focus on fluid animations, modern aesthetics, and intuitive navigation.

---

## ✨ Key Features

- **🍔 Food Delivery**: Browse local restaurants with high-quality imagery and a seamless cart experience.
- **🛒 Grocery Shopping**: Fresh produce and daily essentials delivered at the tap of a button.
- **🚗 Ride Booking**: Safe, reliable, and instant ride-hailing interface.
- **✨ Fluid UI/UX**: Powered by **Framer Motion** for smooth transitions and **Tailwind CSS 4.0** for a modern design system.
- **📱 Fully Responsive**: Optimized for everything from mobile devices to 4K displays.
- **🛒 Real-time Cart**: Persistent context-based cart management.

---

## 🎨 Design Philosophy

GharTk uses a **Premium Display** aesthetic:
- **Color Palette**: Vibrant brand oranges, deep charcoal texts, and soft-shadowed cards.
- **Typography**: Display-focused headings with clean, sans-serif body text.
- **Interactivity**: Staggered entrance animations, hover-lift effects, and responsive feedback.

---

## 🛠️ Tech Stack

- **Core**: React 19 (Hooks, Context, Suspense)
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4.0 (Custom configuration)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router DOM v7

## 🚀 Getting Started

### Prerequisites
- Node.js (Latest LTS)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### Building for Production
```bash
npm run build
```
